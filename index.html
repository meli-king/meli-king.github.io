RISC-V Unit Tester

May 2022 - August 2022 | Utilizes Java, JUnit, Assembly (RISC-V and MIPS)

Together with a group of three other students working under Professor Andrew Petersen, we created a program to test student submitted assembly code automatically. Based off of MIPSUnit, a unit testing framework for MIPS assembly.

In this, I learned how to simulators work, how to work with a large, preexisting repository of code, and how to effectively communicate with multiple people as a team.

Linux Shell Program

January 2022 - April 2022 | Utilizes C, Python, pytest

A C program containing most of the basic functionality of a shell like Bash. This shell can handle simple commands like echo and variable declaration, while also being able to chain commands via pipes and send signals and text to other shell instances. Python and pytest was used to test the correctness of the shell.

In this, I learned about the behaviour of a shell, how use system calls and how to build a large project from the ground up.

Simon

March 2022 | Utilizes Assembly (RISC-V)

The original game of Simon, but made entirely in assembly. UI provided through Ripes, a visual computer architecture simulator and assembly code editor.

In this, I learned how to code in assembly, use system calls and utilize functions that control external devices.

Three Musketeers
